Texto marcado

Podemos criar também um texto marcado usando a tag MARK o texto fica marcado em amarelo

Podemos criar também um texto marcado de outra cor usando a tag MARK com STYLE do CSS posso colocara marcação na cor que eu quiser

Coloquei uma tag de CSS abaixo da tag tittle no head do código, e essa tag irá colocar todas as tags mark que estiver no código sem a formatação CSS na cor que eu definir neste style, neste caso coloquei a cor azul que vai colocar a marcação que estava em amarelo na cor azul, o mark em verde continua verde pois coloquei outro style nele em verde.( Será útil quando tiver muitas marcações no código )
